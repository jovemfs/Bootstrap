Yokais

Layout desenvolvido com bootstrap 4

Esse é um site desenvolvido com Bootstrap como uma das atividades para o Instituto PROA. Apresentando ilustrações sobre yokais - os famosos demônios japoneses -, utilizo da tecnologia Bootstrap para a construção da interface.

Construído com facilidade

Bootstrap é uma biblioteca front-end gratuita e open source com design baseado em HTML e CSS. Com isso, a criação de interfaces torna-se simples e rápida, além de esteticamente agradáveis.

Yokais e tecnologia mágica

Yokai é o nome dado a qualquer tipo de criatura do folclore japonês: demônios, monstros, espíritos, animais com poderes sobrenaturais, objetos com vida, etc. Há tantas lendas sobre yokai no Japão que é quase impossível dizer com precisão quantos deles existem no total.

Mas mais impressionante que os yokais, é o Bootstrap! Nele existe uma funcionalidade incrível chamada "responsividade". Redimensione a janela do navegador para ver a magia tão incrível quanto os antigos mitos japoneses. Com o tráfego móvel - através de celulares - tornando-se mais da metade do tráfego total da internet, a responsividade é um recurso essencial.

Yokais famosos

UMIBOZU

De cor negra e careca reluzente, contam as lendas que o Umibozu é o temível monstro marinho do folclore japonês. a principal ação desse monstro é destruir. Ele sempre aparece para devastar embarcações, levando consigo os seres humanos para as profundezas do mar.

ONAMAZU

O peixe gato gigante é um bagre lendário que dizem ser guardado pelo deus Kashima, que coloca uma pedra mágica gigante em sua cabeça para mante-lo sob controle. Quando deus kashima se distrai de sua vigília ou cai no sono, onamazu se debate, causando terremotos terríveis com a sua cauda.

WANYUDOU

Segundo a lenda, Wanyūdō era um senhor feudal tirano que arrastava suas vítimas até a morte amarradas em uma carruagem ou carro de boi. Como punição, ele adquiriu a forma de uma roda em chamas com seu rosto preso no meio.

Agradecimentos e fontes

Esse projeto foi desenvolvido com ajuda do excelente tutorial disponibilizado por Drew Ryan no YouTube. As informações trazidas sobre os yokais foram retiradas do site Japão em Foco.

Esse é o primeiro de mais três estudos de interface utilizando o Bootstrap. Utilizo de tutoriais como base, mas sempre mudando o CSS e personalizando com algum conteúdo interessante - dessa vez usei Yokais. O foco dos estudos de interface é sempre inovar, nem que seja minimamente, e por isso mudanças no código sempre ocorrem!

Obrigada se você leu até aqui. Um forte abraço e até a próxima modificação!
